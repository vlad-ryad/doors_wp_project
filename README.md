# **ZHOVNER — Магазин дверей на WordPress**

**Сайт:** [u196614.test-handyhost.ru](http://u196614.test-handyhost.ru/)

---

## **О проекте**

Интернет-магазин межкомнатных дверей, созданный с нуля на WordPress с кастомной темой.

---

## **Стек**

*   **Frontend:** HTML5, CSS3, JavaScript
*   **Backend:** PHP, WordPress
*   **Плагины:** Custom Field Suite (CFS), Contact Form 7
*   **Библиотеки:** List.js

---

## **Ключевые функции**

*   Каталог товаров с фильтрацией
*   Динамические карточки товаров
*   Формы обратной связи
*   Адаптивный дизайн
*   Кастомная тема WordPress
